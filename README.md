📌 Project Overview

This project focuses on analyzing student academic performance using structured data analysis techniques. The goal is to uncover patterns, relationships, and key factors that influence student success through data-driven insights and visual storytelling. The analysis combines SQL for data extraction and Python for data analysis and visualization, making it a complete end-to-end data analytics project.

🎯 Goal

To evaluate how different factors such as study habits, attendance, internet access, and demographics influence student academic performance.

🎯 Objectives
- Identify the relationship between study hours and academic performance
- Analyze the impact ofattendance on student academic scores
- Evaluate how internet access affects learning outcomes
- Compare performance across gender and schools
- Derive actionable insights to improve student success

📂 Dataset Description / Collection
The dataset contains student-level academic and behavioral information, including:
- Study Hours  
- Attendance  
- Gender  
- School  
- Internet Access  
- Average Scores  

📥 Data Collection Method
Data was collected and queried using SQL, ensuring structured extraction and efficient handling of records.

🛠️ Tools & Technologies
- SQL – Data extraction and querying  
- Python (Pandas) – Data cleaning and manipulation  
- Matplotlib & Seaborn – Data visualization  
- Jupyter Notebook– Analysis workflow
- 
🔄 Project Workflow
1. Data Collection (SQL)
- Extracted relevant data using SQL queries  
- Filtered and structured dataset for analysis  

2. Data Loading
- Imported dataset into Python using Pandas  
- Verified structure, data types, and completeness  

3. Data Cleaning
- Handled missing values  
- Removed inconsistencies and duplicates  
- Standardized column formats  

 4. Exploratory Data Analysis (EDA)
- Performed univariate and bivariate analysis  
- Identified trends, distributions, and relationships  

5. Data Visualization
- Scatter plots for continuous relationships (e.g., study hours vs score)  
- Bar charts for categorical comparisons (e.g., gender, school)  
- Clean, labeled, and interpretable visual outputs  

📊 Key Insights

- 📈 Study Hours vs Performance  
  Strong positive correlation — more study time leads to higher scores  

- 🌐 Internet Access  
  Students with internet access perform better on average  

- 🏫 School Comparison 
  Performance varies across schools, indicating differences in resources or teaching quality  

- 📅 Attendance- Higher attendance generally improves performance but is not the sole factor  

- 👩‍🎓 Gender Analysis  
  Minimal difference in performance between genders — not a significant factor  

⚠️ Limitations

- Dataset size may limit generalization of findings  
- Lack of additional variables (e.g., socioeconomic background, teaching quality)  
- Analysis is primarily correlation-based, not causation  
- No advanced statistical or machine learning models applied  

🚀 Future Improvements

- Apply statistical testing (correlation coefficient, hypothesis testing)  
- Build predictive models (regression, machine learning)  
- Create interactive dashboards (Power BI / Tableau)  
- Incorporate additional real-world variables for deeper insights  

📌 Conclusion
Student performance is primarily influenced by behavioral factors such as study habits and access to learning resources, rather than demographic attributes like gender.
Improving study patterns, ensuring access to digital resources, and enhancing school-level support systems can significantly boost academic outcomes.
